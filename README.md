
# ESP32-C3 Matter — Modular Firmware Collection

Проект собирается **только под ESP-IDF** (без PlatformIO). Исходники лежат в каталоге `main/`.

## Концепция
Проект представляет собой набор прошивок для ESP32-C3 под Matter.
Тип устройства и состав периферии выбираются **до компиляции** через профиль сборки.

После прошивки:
- настройка Wi-Fi производится через web-интерфейс,
- устройство добавляется в Matter-сеть,
- логика и модель устройства остаются фиксированными.

## Подход
- Сборка: **чистый ESP-IDF** (idf.py), без PlatformIO
- Конфигурация через profiles/*.h
- Matter endpoint'ы создаются на старте и не меняются динамически

## Сборка и прошивка (ESP-IDF)

```bash
# Один раз: задать цель и применить sdkconfig.defaults
idf.py set-target esp32c3

# Сборка
idf.py build

# Прошивка и монитор (подставь свой порт)
idf.py -p COM4 flash monitor
```

Переменная окружения `IDF_PATH` должна указывать на корень ESP-IDF (например, после `export.ps1` / `export.sh` из установки ESP-IDF).

### Если CMake выдаёт «define_property command is not scriptable»

Такое бывает с **CMake 3.30+** при разрешении зависимостей (Component Manager). Решение: использовать CMake **3.28** или **3.29**.

- В системе: установи CMake 3.28/3.29 и выбери его в PATH перед запуском `idf.py build`.
- В VS Code (расширение ESP-IDF): в настройках укажи путь к CMake 3.28 (например, `C:\Users\...\.espressif\tools\cmake\3.28.x`), если он установлен через ESP-IDF Tools.

После смены версии CMake выполни заново:

```bash
idf.py fullclean
idf.py set-target esp32c3
idf.py build
```

## Планируемые типы устройств
- Реле: 1 / 2 / 3 / 4 канала
- PWM: 1 / 2 / 3 канала (+ опциональное реле hard-off)
- Servo: 1 / 2 канала
- Stepper: 1 / 2 канала (3/4 провода, опционально концевики)
- WS2812B: выбор GPIO, количества, порядка RGB
- Возможны комбинированные прошивки

## Web-интерфейс
Web используется только для:
- настройки Wi-Fi
- сервисных операций (reset Wi-Fi / reset Matter)

Тип устройства через web не выбирается.

### WS2812 settings page `/ws2812`
Страница `/ws2812` позволяет настраивать параметры модуля светодиодной ленты WS2812 (если он включён в сборку):
- **enable**: включение/выключение модуля.
- **gpio**: номер GPIO для ленты.
- **LED count** (`count`): количество светодиодов \(1..1024\).
- **color_order**: порядок цветов \(RGB,RBG,GRB,GBR,BRG,BGR\).
- **brightness_limit**: лимит яркости в процентах \(1..100\).
- **transition_ms**: базовое время плавного перехода \(0..5000\) мс.
- **frame_ms**: период перерисовки кадров \(10..100\) мс.
- **power_on_effect / power_off_effect**: эффект при включении/выключении \(none|fade|wipe\).
- **effect_duration_ms**: длительность power on/off эффектов \(0..5000\) мс.

Кнопка **Save config** читает текущий `/api/config`, обновляет секцию `modules.ws2812`, отправляет её в `/api/config` и затем вызывает `/api/apply` для немедленного применения настроек.
